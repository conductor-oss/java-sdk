# Music Generation: Compose, Arrange, Produce, Master, Deliver

Five stages of music production: compose a piece based on genre/mood/duration, arrange it for instruments, produce the track, master the audio, and deliver the final output.

## Workflow

```
genre, mood, durationSec
  -> amg_compose -> amg_arrange -> amg_produce -> amg_master -> amg_deliver
```
